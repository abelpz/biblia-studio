# Security

## Supported versions

We support security updates for the **default branch** (`main`) of this repository as development progresses.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, contact the maintainers **privately**:

- Open a **GitHub Security Advisory** for this repository if the feature is enabled: **Security → Advisories → Report a vulnerability**.
- Or email the repository owner via the contact options on their [GitHub profile](https://github.com/abelpz) if you cannot use advisories.

Include:

- Description of the issue and impact
- Steps to reproduce (if safe to share)
- Suggested fix (optional)

We aim to acknowledge reports within a few business days. This project is volunteer-led; timelines depend on maintainer availability.

## Scope

Reports should concern **this repository and its dependencies** as used in Biblia Studio. Issues in upstream services (e.g. Door43, GitHub itself) should be reported to those projects.
